# Quán Ăn Fusion

## Giới thiệu
Mô tả ngắn gọn về dự án:
Dự án Website bán đồ ăn trực tuyến **Quán Ăn Fusion** được xây dựng nhằm giúp người dùng dễ dàng tìm kiếm, đặt món và thanh toán trực tuyến một cách thuận tiện. 

**Ẩm thực Fusion (Fusion Cuisine)** là sự kết hợp hài hòa, tinh tế giữa các nền văn hóa ẩm thực khác nhau trên thế giới. Tại Quán Ăn Fusion, chúng tôi tự hào mang đến:
- Sự kết hợp hoàn hảo giữa nguyên liệu truyền thống Á Đông cùng kỹ thuật chế biến phương Tây.
- Các món ăn đặc trưng như: Phở Bò Beefsteak, Sushi Roll Gà Nướng, Pizza Hải Sản Xốt Thái.

## Mục tiêu
- Xây dựng giao diện thân thiện, hiện đại (tối ưu UX/UI).
- Hỗ trợ đặt món trực tuyến dễ dàng trên cả máy tính và điện thoại (Responsive).
- Quản lý giỏ hàng thông minh.
- Quản lý đơn hàng và thống kê doanh thu hiệu quả.

## Tính năng
- Đăng ký / Đăng nhập
- Xem danh sách món ăn theo danh mục
- Tìm kiếm món ăn
- Thêm vào giỏ hàng, tính tổng tiền
- Thanh toán trực tuyến
- Quản lý tài khoản người dùng
- Trang quản trị (Admin) dành cho chủ quán

## Công nghệ sử dụng
- HTML5
- CSS3 (áp dụng chuẩn BEM)
- Vanilla JavaScript
- Git / GitHub để quản lý mã nguồn và làm việc nhóm

## Cài đặt và chạy dự án

Clone dự án:
```bash
git clone https://github.com/datcc8a-max/Quan-An-Fusion.git
```

Mở thư mục:
```bash
cd Quan-An-Fusion
```

Chạy file:
```bash
index.html
```
*(Hoặc click chuột phải vào file `index.html` và chọn Open with Live Server trên VS Code)*
